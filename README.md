# ABCD List

A simple mobile-first ABCD task app:

- **A / Attack** — top 5 priorities
- **B / Buy** — things to buy
- **C / Call** — people to call
- **D / Do** — backlog / tasks waiting their turn

Features:

- Four-square grid view
- Scroll/list view
- Add and delete tasks
- Done buttons for every category
- Completed task history
- Completed filters by A/B/C/D
- Auto-promotes Do tasks into Attack when Attack has room
- PWA manifest and offline service worker

## GitHub Pages
